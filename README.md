# cocktails

사용한 데이터 셋: [kaggle dataset](https://www.kaggle.com/datasets/ai-first/cocktail-ingredients)

### 기능 명세

1. 칵테일 리스트 데이터를 확인할 수 있어야 한다.
2. 칵테일 상세 데이터를 확인할 수 있어야 한다.
3. graphQL을 활용해서 서버를 구현해야 한다.
